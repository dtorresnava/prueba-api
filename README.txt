CASTELLANO

Para generar el proyecto opté por el uso de composer ejecutando sus comandos en el CMD de windows 10.

Proyecto hecho con PHP usando el framework simfony4 usando el patrón de diseño establecido por este framework.

Para manejar la llamada a la api de twitter he usado la librería j7mbo/twitter-api-php/TwitterAPIExchange.php.

Para probar en código hay que añadir las access token, access token secret, consumer key y consumer secret en los campos habilitados 
de la función launchTwitterRandockApi del controlador RandockApiController.

El desarrollo calculo que ha sido de unas 4 horas contando con el tiempo dedicado a investigar como funciona simfony4 la librería para lanzar la api.


ENGLISH

To generate the project I used composer executing its commands in the windows 10 CMD.

Project made with PHP using the simfony4 framework and using the design pattern established by this framework.

To handle the call to the twitter api I used the j7mbo / twitter-api-php / TwitterAPIExchange.php library.

To test the code you must add the access token, access token secret, consumer key and consumer secret in the fields enabled
of the launchTwitterRandockApi function of the RandockApiController driver.

The development lasted about 4 hours counting with the time dedicated to investigate how simfony4 works the library to launch the api.
